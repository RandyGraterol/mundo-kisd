# Assets de Imágenes - Mundo Kids

Este directorio contiene los recursos visuales de la aplicación.

## Archivos Disponibles

- **logo.svg** - Logo de Mundo Kids (globo terráqueo con "MK")
- **avatar-nino.svg** - Avatar masculino para la pantalla de bienvenida
- **avatar-nina.svg** - Avatar femenino para la pantalla de bienvenida

## Formato

Todos los assets están en formato SVG para garantizar:
- Escalabilidad sin pérdida de calidad
- Tamaño de archivo pequeño
- Compatibilidad con todos los navegadores modernos

## Personalización

Puedes reemplazar estos archivos SVG con tus propias imágenes (PNG, JPG, SVG) manteniendo los mismos nombres de archivo.

## Uso en la Aplicación

Los assets se cargan desde las rutas:
- `/images/logo.svg`
- `/images/avatar-nino.svg`
- `/images/avatar-nina.svg`
